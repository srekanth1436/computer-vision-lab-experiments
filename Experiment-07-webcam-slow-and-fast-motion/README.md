# Experiment 07: Webcam Slow and Fast Motion

## Aim

Capture webcam video and display smaller and larger views.

## Input

`input/Live webcam`

## Program

The complete runnable implementation is in [`program.py`](program.py).

## Run

From the repository root:

```bash
python Experiment-07-webcam-slow-and-fast-motion/program.py
```

This experiment requires a working webcam and desktop GUI.

## Output

Generated files are saved in the [`output`](output) folder.

## Result

Thus, the program is implemented and executed successfully using Python and OpenCV.
