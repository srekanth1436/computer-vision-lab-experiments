# Experiment 38: Face Detection

## Aim

Detect faces using OpenCV Haar cascades.

## Input

`input/face_sample.jpg`

## Program

The complete runnable implementation is in [`program.py`](program.py).

## Run

From the repository root:

```bash
python Experiment-38-face-detection/program.py
```

For live webcam mode:

```bash
python Experiment-38-face-detection/program.py --webcam
```

## Output

Generated files are saved in the [`output`](output) folder.

## Result

Thus, the program is implemented and executed successfully using Python and OpenCV.
